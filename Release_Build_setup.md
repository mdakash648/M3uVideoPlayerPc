# M3u Video Player PC - Release Build & Setup Guide

এই গাইডটি অনুসরণ করে আপনি খুব সহজেই সোর্স কোড থেকে প্রজেক্টটির রিলিজ বিল্ড (Release Build) এবং উইন্ডোজের জন্য ইন্সটলার (Setup.exe) তৈরি করতে পারবেন।

## ধাপ ১: নতুন রিলিজ ফোল্ডার তৈরি করা
1. আপনার ক্লোন করা প্রোজেক্টের মেইন ফোল্ডারে (যেখানে `CMakeLists.txt` আছে) যান। (উদাহরণস্বরূপ: `d:\c++ project\M3uVideoPlayerPc`)
2. সেখানে নতুন একটি ফোল্ডার তৈরি করুন এবং এর নাম দিন **`M3uVideoPlayerPc_Release`**।

## ধাপ ২: Qt Creator থেকে রিলিজ বিল্ড (Release Build) করা
1. প্রোজেক্টটি Qt Creator-এ ওপেন করুন।
2. বাম দিকের প্যানেল থেকে **Projects** (রেঞ্চ/🔧 আইকন) অপশনে ক্লিক করুন।
3. **Build Settings**-এর নিচে **Edit build configuration** ড্রপডাউন থেকে **Release** সিলেক্ট করুন। (যদি না থাকে, তবে **Add** বাটনে ক্লিক করে তৈরি করে নিন)।
4. এবার প্রোজেক্টটি বিল্ড করুন (`Ctrl + B` চেপে অথবা বাম দিকের নিচে থাকা হাতুড়ি 🔨 আইকনে ক্লিক করে)।
5. বিল্ড শেষ হলে আপনার Qt-এর রিলিজ ফোল্ডারে (সাধারণত প্রোজেক্ট ফোল্ডারের বাইরে `build-release` বা `build/Desktop_Qt_..._Release` নামে তৈরি হয়) যান।
6. সেখান থেকে নতুন তৈরি হওয়া **`M3uVideoPlayerPc.exe`** ফাইলটি কপি করে আপনার প্রোজেক্টের ভেতরের **`M3uVideoPlayerPc_Release`** ফোল্ডারে পেস্ট করুন।

## ধাপ ৩: Windeployqt দিয়ে প্রয়োজনীয় DLL ফাইলগুলো আনা
অ্যাপটি অন্য যেকোনো পিসিতে চালানোর জন্য প্রয়োজনীয় সব লাইব্রেরি (DLLs) ফোল্ডারে আনতে হবে।
1. আপনার কম্পিউটারের Windows Start Menu-তে গিয়ে সার্চ করুন **Qt 6.11.1 (MinGW 64-bit)** (বা আপনার ইন্সটল করা Qt-এর ভার্সন) এবং Command Prompt-টি ওপেন করুন।
2. কমান্ড প্রম্পটে `cd` কমান্ড ব্যবহার করে আপনার তৈরি করা রিলিজ ফোল্ডারে প্রবেশ করুন। উদাহরণস্বরূপ, যদি আপনার প্রোজেক্টটি D ড্রাইভে থাকে:
   ```cmd
   cd /d "d:\c++ project\M3uVideoPlayerPc\M3uVideoPlayerPc_Release"
   ```
3. এরপর নিচের কমান্ডটি কপি করে পেস্ট করুন এবং Enter দিন:
   ```cmd
   windeployqt --qmldir "..\src" M3uVideoPlayerPc.exe
   ```
4. কমান্ডটি শেষ হলে দেখবেন আপনার ফোল্ডারটিতে অ্যাপ রান করার জন্য প্রয়োজনীয় অনেকগুলো নতুন `.dll` ফাইল অটোমেটিক যুক্ত হয়ে গেছে।

## ধাপ ৪: Inno Setup দিয়ে Installer তৈরি করা
1. আপনার কম্পিউটারে [Inno Setup](https://jrsoftware.org/isinfo.php) ইন্সটল করা না থাকলে ইন্সটল করে নিন।
2. প্রোজেক্ট ফোল্ডার থেকে `installer.iss` ফাইলটি Inno Setup দিয়ে ওপেন করুন।
3. উপরের মেনু থেকে **Compile** (অথবা Run > Compile) বাটনে ক্লিক করুন। 
4. কম্পাইল শেষ হলে প্রোজেক্টের ভেতরের `Installer` ফোল্ডারে `M3uVideoPlayer_Setup.exe` নামের নতুন সফটওয়্যার ইনস্টলারটি তৈরি হয়ে যাবে!
